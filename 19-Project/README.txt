For best results, please run the site on a Chrome browser through VS Code Live Server, for example.

First:
Install the free Chrome extension, Moesif Orign & CORS Changer which can be found here:
https://chrome.google.com/webstore/detail/moesif-orign-cors-changer/digfbfaphojjndkpccljibejjbppifbc?hl=en-US

Then:
Once installed on your browser, open Advanced Settings, set site access to “All Sites”, exit and then switch the extension ON.

Then:
Run the site on your Chrome Browser.

You may remove the extension when no longer needed.
